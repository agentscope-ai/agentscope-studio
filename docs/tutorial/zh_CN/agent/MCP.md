# MCP 服务器管理

Friday支持用户动态管理 MCP 服务器
- 可以添加及删除 MCP 服务器
- 可以随时开启和关闭指定 MCP 服务器
- 支持远程和本地两种类型 MCP 服务器


## 添加 MCP 服务器
1.在Friday对话界面有边框点击MCP按钮打开MCP服务器管理界面
![MCP 配置按钮](assets/mcp_botton.png)

2.写 MCP 服务器配置，点击保存即可
![MCP 配置界面](assets/mcp_config.png)

## 开启及关闭 MCP 服务器
在使用过程中，可以通过此开关开启及关闭指定 MCP 服务器
![MCP 开关按钮](assets/mcp_switch.png)
